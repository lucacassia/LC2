### Compile
Once you installed all dependencies you need to compile and run the applications: 

```bash
# Get it from GitHub
git clone https://github.com/megadoro/LC2.git

# Build it
cd openGL/softcore

# Compile
make

# Have fun!
./2DViewer
```
